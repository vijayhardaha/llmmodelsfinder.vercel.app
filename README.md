# LLM Finder

> **Search, compare, and discover AI/LLM models across providers.**
> Visit the live site at [findllm.vercel.app](https://findllm.vercel.app)

LLM Finder is a searchable directory of AI language models. Compare models from OpenAI, Anthropic, Google, Meta, Mistral, DeepSeek, xAI, and more — filtering by pricing, context window, reasoning capabilities, tool support, multimodal input/output, and free availability.

### Features

- **Search** — Find models by name or ID with instant results
- **Filter** — Narrow by provider, family, cost, context size, knowledge cutoff, release year, and capabilities (reasoning, tool calling, structured output)
- **Sort** — Sort any column: price, context length, modality support, etc.
- **Compare** — View pricing per 1M tokens (input, output, reasoning, cache) side by side
- **Discover** — Browse popular searches and learn about model families

Data is sourced from [models.dev](https://models.dev) and refreshed on every visit.
